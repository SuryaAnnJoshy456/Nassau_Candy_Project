\# Nassau Candy Distributor Dashboard



\## Project Overview



This project analyzes the profitability and margin performance of Nassau Candy Distributor products using Python and Streamlit.



\## Features



\* Data Cleaning

\* Exploratory Data Analysis (EDA)

\* Product Profitability Analysis

\* Division Performance Analysis

\* Pareto Analysis (80/20 Rule)

\* Interactive Streamlit Dashboard



\## Technologies Used



\* Python

\* Pandas

\* NumPy

\* Matplotlib

\* Plotly

\* Streamlit

\## Installation



```bash

pip install -r requirements.txt

```



\## Run the Dashboard



```bash

streamlit run app.py

```
